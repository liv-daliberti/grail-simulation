GPT-4o Tools API
================

Utilities under the :mod:`gpt4o` package support evaluating GPT-4o
recommendation slates and interacting with the model provider.  The
modules below expose the main entry points for running evaluations,
preparing conversations, and rendering leaderboard titles.
